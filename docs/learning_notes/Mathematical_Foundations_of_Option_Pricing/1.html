    1. 无套利价格，远期与期权的一般性质

    Xiaotian13 2021/10/16 量化金融

    # 一、无套利价格的一般性质

    市场风险由概率测度 P\mathbb{P}P 确定。

    假设我们有 NNN 个可交易资产（tradeable assets），并且各自拥有初始价格（time-0 prices）

    X0:=(X01,⋯,X0N)\mathbf{X}_0 := \left(X_0^1, \cdots, X_0^N\right) X0​:=(X01​,⋯,X0N​)

    和终值价格（time-TTT prices/payoffs）

    XT:=(XT1,⋯,XTN).\textbf{X}_T := \left(X_T^1, \cdots, X_T^N\right). XT​:=(XT1​,⋯,XTN​).

    我们先研究一阶段模型（one-period model），即我们的时间点只包含time-0和time-TTT。接下来讨论的性质同样也适用于多阶段面模型和连续时间模型。

    # （1）example: 可交易资产

    • 时间 TTT 到期的零息债券（zero-coupon bond）或贴现债券（discount bond）：

      每单位债券在时间 TTT 固定付息一单位，即 ZT=1Z_T = 1ZT​=1.

    • 一个无股息的股票（non-dividend-paying stock）：

      每单位在时间 ttt 有价格 St≥0.S_t \geq 0.St​≥0.

    • 一个银行账户（bank account）：

      每单位在时间 ttt 有价格

      Bt:=exp⁡(∫0trudu),B_t := \exp \left(\int_0^t r_u \text d u \right), Bt​:=exp(∫0t​ru​du),

      其中 rur_uru​ 为随机变量，被称为 time-uuu instantaneous spot rate of interest 或 short rate。需要指出的是，BBB 是通过求解微分方程

      dBtdt=rtBt,with B0=1.\frac{\text d B_t}{\text d t} = r_tB_t, \text{with} \ B_0 = 1. dtdBt​​=rt​Bt​,withB0​=1.

    # （2）静态投资组合与套利

    Definition.\textbf{Definition.}Definition. 一个静态投资组合（static portfolio），是一个数量向量（vector of quantities）：

    Θ:=(θ1,⋯,θN),\Theta := \left(\theta^1, \cdots, \theta^N\right), Θ:=(θ1,⋯,θN),

    其中 θi\theta^iθi 是常数，代表对资产 iii 的投资数量（i=1,⋯,Ni=1,\cdots,Ni=1,⋯,N）。

    如果 θn>0\theta^n > 0θn>0，我们称这个投资组合是做多资产 nnn 的（the portfolio is long asset nnn）；

    如果 θn<0\theta^n < 0θn<0，我们称这个投资组合是做空资产 nnn 的（the portfolio is short asset nnn）。

    一个静态投资组合的time-TTT value 可被计算为

    VT:=Θ⋅X=θ1XT1+⋯+θNXTN.V_T := \Theta \cdot \mathbf X = \theta^1X_T^1 + \cdots + \theta^NX_T^N. VT​:=Θ⋅X=θ1XT1​+⋯+θNXTN​.

    不对 value 和 price 做区分，认为是同一个东西。


    Definition.\textbf{Definition.}Definition. Arbitrage is a combination of transactions that tries to proﬁt from price inconsistencies.

    我们通常所说的套利，并不一定是无风险的。


    Definition.\textbf{Definition.}Definition.

    • 一个静态投资组合 Θ\ThetaΘ ，如果它的价值 VVV 满足

      {V0=0,P(VT≥0)=1,P(VT>0)=0,\left\{ \begin{aligned} &V_0 = 0, \\ &\mathbb P(V_T \geq 0) = 1, \\ &\mathbb P(V_T > 0) = 0, \\ \end{aligned} \right. ⎩⎨⎧​​V0​=0,P(VT​≥0)=1,P(VT​>0)=0,​

      则我们称它是type 1套利。

      即，初始投资值为0，没有亏损的风险，有一定的概率获得收益。

    • 一个静态投资组合 Θ\ThetaΘ ，如果它的价值 VVV 满足

      {V0<0,P(VT≥0)=1,\left\{ \begin{aligned} &V_0 < 0, \\ &\mathbb P(V_T \geq 0) = 1, \\ \end{aligned} \right. {​V0​<0,P(VT​≥0)=1,​

      则我们称它是type 2套利。

      即，初始获得一笔钱，并且最后一定不需要再还了。

    V0V_0V0​ 代表最开始投入的钱，因此如果 V0<0V_0 <0V0​<0，即代表最开始获得钱；VTV_TVT​ 代表投资组合最后能卖出的钱，因此如果 VT≥0V_T \geq 0VT​≥0，即代表投资组合最后一定不需要再投入新的钱了。

    • 一个静态投资组合 Θ\ThetaΘ，如果它是"type 1"套利或者"type 2"套利，我们则称它为套利（arbitrage）。

    因此，这里的“套利”指的就是无风险套利。

    我们在对资产做定价时，实际上就是要找一个没有套利空间的价格（arbitrage-free price）。

    Example.\mathbf{Example}.Example. 假设资产 F,G,HF, G, HF,G,H 满足：FT≥GT+HT,F0<G0+H0F_T \geq G_T + H_T, F_0 < G_0 + H_0FT​≥GT​+HT​,F0​<G0​+H0​，则 （1 unit of FFF, -1 unit of GGG, -1 unit of HHH）是type 2套利。

    # （3）The law of one price

    Proposition.\textbf{Proposition}.Proposition. 假设投资组合 Θa\Theta ^aΘa 是 Θb\Theta^bΘb 的上位替代（super-replicates），即满足 P(VTa≥VTb)=1\mathbb P(V_T^a \geq V_T^b)=1P(VTa​≥VTb​)=1，那么一定有 V0a≥V0bV_0^a \geq V_0^bV0a​≥V0b​，不然就存在套利空间。

    Proof.\textbf{Proof}.Proof. 假设有 V0a<V0bV_0^a < V_0^bV0a​<V0b​，则构造投资组合 Θ:=Θa−Θb\Theta := \Theta^a - \Theta^bΘ:=Θa−Θb，则有 V0=V0a−V0b<0,VT=VTa−VTb≥0V_0 = V_0^a - V_0^b < 0, V_T = V_T^a - V_T^b \geq 0V0​=V0a​−V0b​<0,VT​=VTa​−VTb​≥0，属于type 2套利。 □\square□

    同样地，假设投资组合 Θa\Theta ^aΘa 是 Θb\Theta^bΘb 的下位替代（sub-replicates），即满足 P(VTa≤VTb)=1\mathbb P(V_T^a \leq V_T^b)=1P(VTa​≤VTb​)=1，那么一定有 V0a≤V0bV_0^a \leq V_0^bV0a​≤V0b​，不然就存在套利空间。根据这两点，我们可以得出重要的定价法则：

    Proposition.\textbf{Proposition}.Proposition. 如果 P(VTa=VTb)=1\mathbb P(V_T^a = V_T^b) = 1P(VTa​=VTb​)=1，那么有 V0a=V0bV_0^a = V_0^bV0a​=V0b​。

    这就是the low of one price，即如果我们想知道一种证券的价格，使用尽可能与其相似的另一种证券（或证券组合）的价格。

    # （4）Price vs Value vs Payoff

    • Time-ttt price = how much it costs to buy/sell something at time ttt. (Exceptions: "forward price", "futures price")

    • Time-t value = how much it should cost to buy/sell something.

      Meaning of “should” depends on the context. Here, the only notion of “should” is that arbitrage should not exist. So for us, "value" is what it costs to buy/sell something, in the absence of arbitrage. But since we have a standing assumption of no-arbitrage, we really have no distinction between price and value, unless we are in a situation where arbitrage exists.

    • Payoﬀ = Payout = how much a contract pays = Value of the contract at expiration (assuming single payment)

    # （4）总结

    Using static positions in TTT-expiry bonds, forwards, calls, and puts on SSS, we can replicate or bound (super-replicate, sub-replicate) general functions of STS_TST​.

    • Use bonds to adjust level.（截距）
    • Use forwards (or perhaps SSS itself) to adjust slope.（斜率）
    • Use calls (or puts) to adjust convexity and concavity.（凹凸性）
