    # 一、矩阵范数和从属范数

    Definition3.1.1.\textbf{Definition 3.1.1.}Definition 3.1.1. 我们称 Kn↦R+\mathbb K^n \mapsto \mathbb R^+Kn↦R+，满足下列条件的映射为 范数（norm） ，用记号 ∥⋅∥\|\cdot\|∥⋅∥ 表示：

    2. ∀x∈Kn,∀λ∈K,∥λx∥=∣λ∣∥x∥\forall x \in \mathbb K^n, \forall \lambda \in \mathbb K, \quad \|\lambda x\| = |\lambda|\|x\|∀x∈Kn,∀λ∈K,∥λx∥=∣λ∣∥x∥（Homogeneity）；
